# AISVC-midi-web 开发检验文档

> 每个功能模块的检验用例、验收标准、检验手段
>
> 检验手段缩写：
> - 👁 **肉眼** = 用户肉眼审查 UI 渲染是否正确
> - ✋ **手操** = 用户手动操作，验证交互逻辑是否符合预期
> - 🤖 **自测** = 我编写测试脚本/数据，调 API 或模拟状态变更，验证结果
> - 🔊 **听感** = 播放音频，耳朵验证

---

## 检验节奏

每完成一个 Phase，跑完该 Phase 的全部检验用例，全部 ✅ 后再进入下一 Phase。
